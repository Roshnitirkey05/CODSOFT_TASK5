📒 Advanced Contact Book – Python Tkinter

A GUI-based Contact Book application built using Python and Tkinter.
This application allows users to add, edit, delete, search, and manage contacts with an easy-to-use graphical interface. Contacts are stored locally using a JSON file for persistent data storage.


✨ Features

- ➕ Add new contacts
- ✏️ Edit existing contacts
- ❌ Delete contacts
- 🔎 Search contacts by name
- 📋 Display all saved contacts
- 🖼 Upload profile photo for contacts
- 💾 Data saved using JSON file storage
- 🖥 Clean and user-friendly GUI



🛠 Technologies Used

- Python
- Tkinter (GUI Library)
- JSON (Data Storage)
- OS Module
- File Dialog & Messagebox



📂 Project Structure

contact-book/
│
├── contact_book.py
├── contacts.json
└── README.md




🎯 How It Works

1. Enter a Name and Phone Number
2. Click Add to save the contact
3. Select a contact from the list to Edit or Delete
4. Use Search to find contacts quickly
5. Contacts are automatically stored in contacts.json
